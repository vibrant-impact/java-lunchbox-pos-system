
# 🍱 Lunchbox POS System
**A Multi-Item Order Tracker & Reward Game**

### 🌟 Project Overview
Created as part of my Java development studies, this application simulates a professional Point of Sale (POS) system with a focus on clean UI in the console and robust Object-Oriented logic.

### ✨ Key Features
* **Multi-Item Support:** Unlike standard assignments, this allows multiple items per customer bag.
* **Casino-Style Rewards:** A 1-in-5 chance to win a free random snack using `SecureRandom`.
* **Soft-Delete System:** Orders are "Cancelled" rather than deleted to maintain a data trail.
* **Visual Formatting:** Custom `printf` grid menus and double-border receipts.

### 🛠️ Tech Stack
* **Language:** Java
* **Design Pattern:** Object-Oriented (Encapsulation & State Management)
* **Tools:** VS Code, Git/GitHub
